# 🎲 Yahtzee Telegram Bot

Telegram-бот для игры в классические кости Yahtzee.

## Особенности

- **Мультиплеер**: Поддержка до 4 игроков в одной игре
- **Интерактивная таблица**: Красивое отображение очков
- **Умный подсчёт**: Автоматический расчёт бонусов и итоговых сумм
- **Защита от ошибок**: Проверка правил и последовательности ходов
- **Удобный интерфейс**: Все действия через команды Telegram

## Правила игры

Yahtzee — это классическая игра в кости, где игроки по очереди бросают 5 кубиков, пытаясь набрать максимальное количество очков в 13 категориях.

### Ход игры:
1. **Бросок**: Игрок бросает все 5 кубиков (`/roll`)
2. **Переброс**: Можно перебросить любые кубики до 2 раз (`/reroll 1 3 5`)
3. **Запись очков**: Выбрать одну из 13 категорий (`/score 7`)

### Категории очков:

#### Верхняя секция:
- **1-6**: Сумма соответствующих значений на кубиках
- **Бонус**: +35 очков если сумма верхней секции > 63

#### Нижняя секция:
- **7. Три равных**: Сумма всех кубиков (минимум 3 одинаковых)
- **8. Четыре равных**: Сумма всех кубиков (минимум 4 одинаковых)
- **9. Фул-хаус**: 25 очков (3 одинаковых + 2 одинаковых)
- **10. Малая последовательность**: 30 очков (4 последовательных значения)
- **11. Большая последовательность**: 40 очков (5 последовательных значений)
- **12. Yahtzee**: 50 очков (все 5 кубиков одинаковые)
- **13. Шанс**: Сумма всех кубиков (любая комбинация)

## Установка и запуск

### Требования
- **Ruby** 3.3 или выше
- **Gem** `telegram-bot-ruby`

### Установка

1. **Клонируйте репозиторий:**
git clone https://github.com/yourusername/yahtzee-telegram-bot.git

cd yahtzee-telegram-bot

3. **Установите зависимости:**
gem install telegram-bot-ruby

4. **Создайте бота в Telegram:**

- Найдите @BotFather в Telegram
- Отправьте команду /newbot
- Следуйте инструкциям для создания бота
- Сохраните полученный токен

4. **Настройте токен:**

- Откройте файл **yahtzee_bot.rb**
- Замените **YOUR_BOT_TOKEN_HERE** на ваш токен:

  TOKEN = 'YOUR_BOT_TOKEN_HERE'

5. **Запустите бота:**
ruby bot.rb
